# deploy-market

#### Command

```text
node scripts deploy-market
```

#### Parameters

| Name | Type | Message |
| :--- | :--- | :--- |
| **address** | `string` | _Contract deploy address_ |
| media\_contract | `string` | _Address of the NFT/Media contract_ |

{% hint style="info" %}
Look into the [Market](../contracts/market.md) contract for types and methods.
{% endhint %}
